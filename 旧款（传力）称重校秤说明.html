旧款（传力）称重校秤说明 Setting Instructions for Old Transducer Weighing Scale

设置方法 Setting Method

步骤 1: 进入设置模式 Step 1: Enter Setting Mode
变送器通电 Power on the transmitter
长按 SET 键 Long press SET key
直到显示 F1 Until display shows F1
进入设置模式 Enter setting mode
步骤 2: 调整设置值 Step 2: Adjust Settings
使用 向下箭头 键 Use ↓ key
进入当前设置 Enter current setting
使用 左/右箭头 键 Use ←/→ keys
改变设置值 Change setting value
短按 SET 键 Short press SET key
保存设置值 Save setting value
步骤 3: 导航 Step 3: Navigation
在主菜单下 Under main menu
使用 左/右箭头 键 Use ←/→ keys
改变设置项 Change settings
长按 向上箭头 键 Long press ↑ key
返回到称重模式 Return to weighing mode
步骤 4: 参数设置 Step 4: Parameter Settings

F1 设为 20000（铅粉）

F1 set to 20000 (lead powder)

水和酸设为 3000

Water and acid set to 3000

F10 设为 0.0（一位小数）

F10 set to 0.0 (one decimal)

F1 × F10 = 量程

F1 × F10 = Range

校秤方法 Calibration Method

步骤 1: 清零当前状态 Step 1: Clear Current Status
进入 F16 Enter F16
按 SET 键 Press SET key
然后返回 Then return
（当前状态清零） (Current status cleared)
步骤 2: 输入砝码重量 Step 2: Enter Weight Value
进入 F17 Enter F17
在称台上放砝码 Put weights on the counter
在 F17 中输入砝码重量值 Enter weight value in F17
按 SET 键 Press SET key
返回到称重模式 Return to weighing mode
校称完成 Calibration is complete
