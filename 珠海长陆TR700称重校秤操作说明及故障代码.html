珠海长陆 TR700 称重校秤操作说明及故障代码

称重校秤操作指南

一、基本操作

面板按键功能说明

面板按键 功能说明 详细说明
MENU 菜单 1）在称重状态下，进入菜单键
2）在菜单设置里，退出键
3）在子菜单项里，不保存参数进入下一子菜单项
GR/NT 毛重/净重 1）在称重状态下，毛重或净重转换
2）在输入数据时左移键
TARE 去皮 1）在称重状态下，去皮键（去皮范围：最大量程的百分之八十）
2）在菜单设置选择上一菜单项
3）在输入数据时数据递增键
FUNC 功能 1）在菜单设置选择下一菜单项
2）在输入数据时为递减键
ZERO 清零 1）在称重状态下，清零键
2）在菜单设置里，确认键
3）在子菜单项里，保存参数进入下一子菜单项

功能设定步骤

在称重显示下，按 MENU 通过 GR/NT TARE FUNC 插入密码，按 ZERO 确认密码进入菜单，按 TARE FunC 选择"Func"，按 ZERO 进入功能参数设定
通过 TARE FUNC 选择相应的参数
按 ZERO 进入参数修改，通过 TARE FUNC 修改参数
按 ZERO 保存此参数，或者按 MENU 不保存此参数，退回到上一级菜单
循环执行第二步到第四步，直到修改完所需参数，按 MENU 退出到"FunC"，此时参数保存到内存中
按 MENU 退出到正常显示接口 “F00---0”

二、校秤

注意事项
  • 标定时不允许进行零位跟踪即 F1 设为 0，而且标定前须要预先半小时开机，使传感器、仪表等称重单元达到热稳定
  • 在标定时，只有当称体稳定时，即重量检测指示灯 motion ⚫灭，才能接受标定。当 motion 🟡灯很长时间没有熄灭，请检查 F2 参数设置
  • 若是输入的参数不正确，则会显示对应错误提示约 2 秒钟后自动回到需要重新输入参数处

校称步骤

步骤1：进入标定菜单 在称重状态下，进入一级菜单，选择 CAL，输入默认密码 00001，进入标定菜单，选择 CAL1
步骤2：设置小数点 按 ZERO 确定，显示 C1.dECi，进入小数点设定，选择 1（1 位小数）
步骤3：设置分度值 按 ZERO 确定，显示 C1.d，设置分度值，选择 1
步骤4：设置最大量程 按 ZERO 确定，显示 C1.MAX，按 ZERO 确定，设置最大量程，铅粉设置 2000.0，水和酸设置 300.0
步骤5：零点标定 按 ZERO 确定，显示 C1.Zero，按 ZERO 进入零点标定，待称体稳定，此时显示 000000 且全部闪烁
步骤6：重量标定 按 ZERO 确定，显示......且全部闪烁，之后显示 C1.SPAn，按 ZERO 进入重量标定，输入重量值，待称体稳定后再按 ZERO 确定，完成标定
重要提示
砝码的重量应当与最大称量值相同或稍轻

三、模拟量

在称重显示下，按 MENU 通过 GR/NT TARE FUNC 输入密码，按 ZERO 确认密码进入菜单，按 TARE FUNC进入 选择 CAL，按 ZERO 显示 CAL1，按 TARE FUNC 选择 Ao-i/Ao-v，按 ZERO 进入第二步

模拟量输出类型

  • Ao-i：模拟量输出是电流型，0-20mA 或者 4-20mA 输出
  • Ao-v：模拟量输出是电压型，0-5V 或者 0-10V 输出
通过 TARE FUNC 选择 CAo.ZEr，CAo.SPA，按 ZERO 进入参数值修改，修改完按 ZERO 保存参数并退出

参数说明

  • CAo.ZEr：零点对应的重量值（正负数都可）。0-20mA 的 0mA/4-20mA 的 4mA/0V 对应的重量值
  • CAo.SPA：满点对应的重量值（正负数都可）。20mA/5V/10V 对应的重量值

五、错误代码

错误代码 说明 解决方法
DISP-L 输入信号<0.05mV或为负信号时:可能接线错误、松动、四线制没有短接EX+和SEN+，或者没有短接EX-和SEN-等 重新调整传感器的接线或者在EX+和SIG+之间增加一个准确度为1%的50k~500K电阻
DISP-H 输入信号>31mV时:可能接线错误、松动、四线制没有短接EX+和SEN+或者没有短接EX-和SEN-等 重新调整传感器的接线或者在EX+和SIG+之间增加一个准确度为1%的50k~500K电阻
Error0 可能接线错误，或者ad转换部分有问题 重新调整传感器接线或更换仪表
Error1 "最大量程/最小刻度"无法整除，显示分辨率大于50000或者显示分辨率小于300，或者最大量程小于100，或者最大量程大于900000，或者是上次灵敏度/这次修改后的分度小于0.3uV/d 调整最大量程、分度间距或者更换传感器或降低标定精度
Error2 零点电压过大，大于15mV 应在EX+和SIG-之间增加一个准确度为1%的50k~500K电阻，或减轻自重
Error3 零点电压过低，低于0.05mV 应在EX+和SIG+之间增加一个准确度为1%的50k~500K电阻
Error4 输入的重量值大于最大量程值 重新输入重量值或者最大量程
Error5 传感器输入的灵敏度过低，小于0.3uV/d或者输入的重量值为0 更换传感器或降低标定精度
Error6 重量标定的毫伏值电压小于零点标定的毫伏值电压，标定时产生的错误操作，或接线错误 重新接线或重新对秤标定
Error7 传感器的输入大于信号最大输入范围31mV 调整传感器的接线或更换传感器
Error8 重量标定时输入的重量值小于100个分度 重新输入重量值
Error9 重量标定"输入的重量值/最小刻度"无法整除 重新输入重量值

六、恢复出厂设置

显示 rESU，按 ZERO 进入参数恢复出厂设定
选择所需要恢复出厂设定值的参数选项，输入正确的系统密码即可把相应的参数恢复出厂设定值
选项 说明
rE-Func 功能设定恢复出厂设定
rE-SEtP 高低限参数恢复出厂设定
rE-CAL 标定参数恢复出厂设定
rE-CoM/CAn/nEt 通讯参数恢复出厂设定（有通讯扩展板才会出现此菜单项）
