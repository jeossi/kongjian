新款（珠海长陆）称重校秤说明 Calibration Instructions for New Zhuhai Changlu Weighing Scale

校秤步骤 Calibration Steps

步骤 1: 进入标定菜单 Step 1: Enter Calibration Menu
在称重状态下 In the weighing state
进入一级菜单 Enter first-level menu
选择 CAL Select CAL
输入密码 00001 Enter password 00001
按确定键 Press OK key
进入标定菜单 Enter calibration menu
步骤 2: 设置小数点 Step 2: Set Decimal Point
选择 CAL1 Select CAL1
按确定键 Press OK key
显示 C1.dECi Display shows C1.dECi
进入小数点设定 Enter decimal point setting
选择 1（1位小数） Select 1 (1 decimal place)
步骤 3: 设置分度值 Step 3: Set Index Value
按确定键 Press OK key
显示 C1.d Display shows C1.d
设置分度值 Set index value
选择 1 Select 1
步骤 4: 设置最大量程 Step 4: Set Maximum Range
按确定键 Press OK key
显示 C1.MAX Display shows C1.MAX
设置最大量程 Set maximum range
铅粉设置 2000.0 Lead powder set to 2000.0
水和酸设置 300.0 Water and acid set to 300.0
步骤 5: 零点标定 Step 5: Zero Calibration
按确定键 Press OK key
显示 C1.Zero Display shows C1.Zero
按确定键进入零点标定 Press OK key to enter zero calibration
待称体稳定，此时显示 000000 且全部闪烁 When the weighing body is stable, display shows 000000 and all flashing
步骤 6: 重量标定 Step 6: Weight Calibration
按确定键 Press OK key
显示......且全部闪烁 Display shows...... and all flashing
显示 C1.SPAn Display shows C1.SPAn
按确定键进入重量标定 Press OK key to enter weight calibration
输入重量值 Enter weight value
待称体稳定后再按确定键，完成标定 Press OK key after the weighing body is stable to complete calibration
